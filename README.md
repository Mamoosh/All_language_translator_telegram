# All_language_translator_telegram
All language translator for Telegram by Python.

How to use :

pip install googletrans==3.1.0a0
pip install telepot

get your bot token from https://t.me/BotFather
change Token and userid

how to get user id ? 
https://bigone.zendesk.com/hc/en-us/articles/360008014894-How-to-get-the-Telegram-user-ID-

Libraries used in this code :

googletrans 3.1.0a0
telepot
